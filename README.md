# Medicare Part D Prescriber Dashboard – 2023

## Overview

This repository contains an interactive healthcare analytics dashboard developed using Tableau and 2023 Medicare Part D Prescriber Public Use File (PUF) data. The project focuses on analyzing prescription utilization, beneficiary distribution, and drug cost trends across the United States.

The dashboard was designed to convert large-scale raw healthcare datasets into meaningful visual insights that support healthcare analytics, public health reporting, and data-driven decision-making.

---

## Project Objectives

The primary objectives of this project were to:

* Analyze Medicare Part D prescribing patterns across the United States
* Visualize total drug cost distribution geographically
* Identify prescribing trends and beneficiary utilization
* Develop interactive KPI dashboards for healthcare analytics
* Enable state-level filtering for detailed regional analysis
* Explore relationships between claims, beneficiaries, and prescription costs

## Dataset Information

The dashboard uses Medicare Part D Prescriber data for 2023, which includes information related to:

* Prescription drug utilization
* Provider prescribing behavior
* Total drug costs
* Claims submitted
* Beneficiary counts
* Drug categories and utilization types

This publicly available dataset provides valuable insight into healthcare utilization and prescription trends across the United States.

---

## Data Engineering Process

### 1. Raw Data Acquisition

The original Medicare Part D dataset was imported from publicly available healthcare data sources.

### 2. Data Cleaning Using SQL

The raw data contained large volumes of records requiring preprocessing and standardization. SQL was used to:

* Remove duplicate records
* Handle null or missing values
* Standardize column formats
* Create normalized analytical tables
* Aggregate measures for dashboard KPIs
* Optimize query performance for visualization

### 3. Data Modeling

Structured tables were created to support efficient dashboard rendering and analytical reporting.

## Dashboard Features

### 📍 Geographic Drug Cost Distribution

* Interactive U.S. map visualization
* Displays Total Drug Cost distribution by state
* Enables regional comparison of prescription spending

### 📊 KPI Summary Section

Key performance indicators included:

* Total Prescribers
* Total Claims
* Total Beneficiaries
* Drug Type Utilization

These KPIs provide high-level insights into prescription activity and healthcare utilization.

### 📈 Scatter Plot Analysis

An interactive scatter plot was developed to analyze relationships between:

* Beneficiary counts
* Total claims
* Drug cost

This visualization helps identify prescribing patterns and cost distribution trends.

### 🎛 Interactive Filters

Dashboard users can dynamically filter data by:

* State
* Drug type
* Other analytical dimensions

This functionality enables detailed exploration and customized analysis.
